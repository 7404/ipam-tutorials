0. MNIST: recognizing handwritten digits


This example demonstrates how to train different models on the MNIST  dataset, using multiple optimization techniques: SGD, ASGD, CG, L-BFGS.


The example demonstrates a classical example of training well-known models: convolutional neural networks (CNNs, or ConvNets), multi-layer neural networks (MLPs), and simple logistic regression units, on a 10-class classification problem. 


The tutorial illustrates several points:


  1. description of the model
  2. choice of a loss function (criterion) to minimize
  3. creation of a dataset as a simple Lua table
  4. description of training and test procedures


Note: the example script is quite verbose, on purpose. Instead of relying on opaque classes, dataset creation, as well as the complete training loop are basically exposed right here. Although a bit challenging at first, it should help new users quickly get independent, and able to tweak the code for their own problems.


1. Description of the model


In this example, we describe three models. To do so, we rely on the ‘nn’ package, which provides a set of basic building (lego) blocks of pre-defined backpropable transfer functions.


The first step is to load the ‘nn’ library:
